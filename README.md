# Job/Internship Application Assistant Agent

An AI agent built for Kaggle's AI Agents Intensive Vibe Coding Capstone.

## What it does
- Parses a resume and job posting
- Generates a tailored cover letter
- Highlights matched skills and gaps
- Tracks applications over time (log, view, update status)
- Understands natural language instructions to decide which action to take

## Tech stack
- Google Gemini API (google-genai SDK)
- Function calling / tool use for natural language agent behavior
- JSON-based persistent memory for application tracking
